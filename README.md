## Trabalho Final - CD1

O presente trabalho tem como objetivo aplicar, avaliar e interpretar o desempenho de três algoritmos de classificação supervisionada em bases de dados reais.
**Bases de dados**
- Base 1 (doença do coração): https://archive.ics.uci.edu/ml/machine-learning-databases/heart-disease/processed.cleveland.data
- Base 2 (churn de clientes): https://raw.githubusercontent.com/IBM/telco-customer-churn-on-icp4d/master/data/Telco-Customer-Churn.csv

**Estrutura do projeto**
- base1/: scripts para exploração, pré-processamento e modelos na base de doença do coração.
- base2/: scripts para exploração, pré-processamento e modelos na base de churn.

**Como executar**
1. Instale as dependências em Python (pandas, numpy, scikit-learn, matplotlib/seaborn).
2. Execute, em cada pasta (base1 ou base2), os arquivos `2_exploracao_preprocessamento.py`, `3_knn.py`, `4_arvore.py` e `5_svm.py` na ordem desejada.
